# Smart Grease Trap Stopper

## 專案介紹
本專案為一個智慧油脂攔截裝置的展示系統，旨在解決餐飲業油脂堵塞與環境污染問題。

## 功能
- 油脂累積監測
- 即時狀態顯示
- 視覺化介面（Web）

## 使用技術
- Python（Streamlit）
- HTML / CSS / JavaScript
